# Portfolio Optimization

This project builds a time series forecasting and portfolio optimization pipeline for GMF Investments using historical data for TSLA, BND, and SPY.

## Setup
1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Run Jupyter: `jupyter notebook`

## Tasks
- **Task 1:** EDA and data preprocessing
- **Task 2:** ARIMA/SARIMA forecasting
- **Task 3:** LSTM forecasting
- **Task 4:** Portfolio optimization (Efficient Frontier)
- **Task 5:** Backtesting
